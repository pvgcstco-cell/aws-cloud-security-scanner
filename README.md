# AWS Cloud Security Scanner

This is an AWS Cloud Security Scanner powered by Prowler. It helps in auditing AWS security configurations by running checks on multiple services like IAM, EC2, S3, etc. It generates a detailed report that can be visualized using a dashboard.

## Features:
- AWS security scan using Prowler.
- Interactive report generated.
- Dashboard for real-time visualization.
